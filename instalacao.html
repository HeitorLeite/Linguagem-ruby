Passo a Passo para Instalar o Ruby

  1. Baixar o Ruby

    Visite o site oficial do Ruby em ruby-lang.org.

  2. Escolher seu Sistema Operacional

    Selecione o link de download apropriado para o seu sistema operacional (Windows, macOS ou Linux).

  3. Instalar no Windows

    • Baixe o instalador do Ruby para Windows.
    • Execute o instalador e siga as instruções na tela.
    • Escolha as configurações padrão para a maioria dos usuários.

  4. Instalar no macOS

    • Use o Homebrew para instalar Ruby. Abra o terminal e execute:
    brew install ruby

  5. Instalar no Linux

    • Abra o terminal.
    • Atualize a lista de pacotes do sistema.
    • Instale o Ruby usando o gerenciador de pacotes do seu sistema.
    # Comandos para Debian/Ubuntu
sudo apt update
sudo apt install ruby
# Comandos para Fedora
sudo dnf install ruby
# Comandos para Arch Linux
sudo pacman -S ruby

  6. Verificar a Instalação

    Abra o terminal e execute ruby -v para verificar se a instalação foi bem-sucedida.

  7. Teste

    Crie um arquivo hello.rb e escreva o código abaixo:

    puts "Hello, world!"

    Execute com ruby hello.rb no terminal. Se a mensagem aparecer, a instalação funcionou.
